LISTA DE EXERCICIOS EM C++

Arquivos:
- Exercicio1.cpp: leitura e exibicao de uma data.
- Exercicio2.cpp: soma de dois numeros usando uma funcao.
- Exercicio3.cpp: conversao de dolares para reais usando funcao inline.
- Exercicio4.cpp: matriz bidimensional de despesas por ano e trimestre.

OBSERVACAO SOBRE O EXERCICIO 3:
Como o enunciado nao informou uma cotacao fixa, o programa solicita a cotacao
atual do dolar ao usuario.

COMO COMPILAR NO TERMINAL DO VS CODE

Windows:
g++ Exercicio1.cpp -o Exercicio1.exe
.\Exercicio1.exe

Linux:
g++ Exercicio1.cpp -o Exercicio1
./Exercicio1

Troque o numero do arquivo para executar outro exercicio.
