LISTA DE EXERCICIOS - POO AVANCADA EM C++

COMO COMPILAR NO VS CODE

Windows:
g++ -std=c++17 Exercicio1.cpp -o Exercicio1.exe
.\Exercicio1.exe

Linux:
g++ -std=c++17 Exercicio1.cpp -o Exercicio1
./Exercicio1

Troque o numero do exercicio conforme o arquivo desejado.
